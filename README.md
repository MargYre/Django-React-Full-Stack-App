# Django-React-Full-Stack-App
Using Python and JavaScript to create a full-stack web application.
Django for the backend and React for the frontend, as well as setting up authentication using JWT tokens.

## Prerequisites
- Python 3.x
- Node.js
- npm or yarn
- Git
